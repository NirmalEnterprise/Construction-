How to publish on GitHub Pages (free):
1. Create a free GitHub account: https://github.com
2. Create a new repository (example: nirmal-enterprises)
3. Upload all project files & folders
4. Go to Settings > Pages → Select branch “main” and folder “/ (root)” → Save
5. Wait a few minutes → Your site will be live at:
   https://<your-github-username>.github.io/<repo-name>/
6. Copy this link and add it to your Instagram bio
